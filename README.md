## 🚀 Installation

- Open the Extensions sidebar in VS Code
- Search for `Bearded`
- Click Install
- Open the Command Palette with Ctrl+Shift+P or ⇧⌘P
- Select Preferences: Color Theme and choose `Bearded Theme`.
